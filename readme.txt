1. npm install
2. log in okta developer console & create api token & client secret &  change all those variables in .env file
3. Register from any username & craete a admin group
4. Take the admin id in console & save it in .env file & use it everywhere wherever group id is being used(e.g : admin route,delete route)
5. Go to 'go to admin' page from dashboard to see all the user registered user in this application
6. Delete/deactivate user from delete user route & it won't show in the admin dashboard
